# Next Dr - Dr appointment Application

## Stack
- React
- Node
- Express
- Mongodb
  


## Features
- JWT authentication and authorization
- Pagination
- Filteration
- Formik validation
- Separate dashboard for every role
- Charts
- Review
- Crud operations

 A complete full stack web application
